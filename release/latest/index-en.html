Restricted Phenotype Classc back to ToC or Class ToC

IRI: https://www.smith.care/cop#Restricted_Phenotype_Class

The extension (set of instances) of a restricted phenotype class is restricted to a certain range of individual phenotypes by defined conditions. For example, the phenotype class ‘young age’ instantiated by the ages of the young living beings (e.g., if the age is below 30 years) is restricted. A restricted phenotype class is a subclass of a non-restricted phenotype class (e.g., ‘young age’ is subclass of 'age').
has super-classes
Phenotype Class c
has sub-classes
Restricted Combined Phenotype Class c, Restricted Derived Phenotype Class c, Restricted Single Phenotype Class c
